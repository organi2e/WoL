# WoL

An implementation to send magick packet for Wake-On-Lan

## Usage

```shell
wol [options] <MAC addresses> …    
```

### Example:    

```shell
wol --addr 192.168.1.1 EA-10-DE-AD-BE-EF  
```

### Options:

```shell
  --port <port number>    
    Default: 9    
  --addr <IPv4 address>    
    Default: broadcast    
  --verbose    
    Verbose flag
```
